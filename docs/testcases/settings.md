# Settings Test Cases

| ID | Title | Description | Expected Result |
|---|---|---|---|
| SET-001 | Change Name | Update display name and save. | New name is reflected in the profile and for other users. |
| SET-002 | Toggle Dark Mode | Select "Dark Mode" in appearance. | App UI colors update immediately. |
| SET-003 | Block Contact | Block a user from privacy settings. | User appears in the blocked list and cannot message the user. |
| SET-004 | Delete Account | Attempt deletion with old session. | App prompts for "Recent Login" re-authentication. |
| SET-005 | Verify Email | Request email change and click link in inbox. | Email status changes from "Pending" to "Verified". |
