# Chat Features Test Cases

| ID | Title | Description | Expected Result |
|---|---|---|---|
| MED-001 | Send Image | Pick image from gallery and send. | Image is uploaded, thumbnail appears in chat. |
| MED-002 | Record Audio | Hold record button for 5s and release. | Audio message is sent and playable. |
| MED-003 | Cancel Record | Hold record and swipe left. | Recording stops and is discarded. |
| MED-004 | Full-screen View | Tap on an image message. | Image opens in full-screen viewer. |
| MED-005 | Audio Playback | Tap play on a received audio message. | Audio plays, slider moves with progress. |
