# Chats Test Cases

| ID | Title | Description | Expected Result |
|---|---|---|---|
| CHT-001 | Send Message | Type "Hello" and tap send. | Message appears in bubble with checkmark. |
| CHT-002 | Receive Message | Receive message while chat is open. | Message appears, read receipt is sent to sender. |
| CHT-003 | Swipe Archive | Swipe left on a conversation. | Conversation disappears from "All" list and moves to "Archived". |
| CHT-004 | Typing Status | Other user starts typing. | "Typing..." indicator appears under their name. |
| CHT-005 | Offline Status | Kill app on other device. | Status changes to "Last seen [Time]" after timeout. |
