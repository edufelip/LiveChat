# Home Screen Requirements

## R-012: Bottom Navigation
The system shall provide a persistent bottom navigation bar with four tabs: Chats, Calls, Contacts, and Settings.

## R-013: Active Tab Indication
The system shall visually highlight the currently active tab.

## R-014: Global Unread Badge
The system shall display a numeric badge on the "Chats" tab icon representing the total count of unread messages.

## R-015: Navigation State Persistence
The system shall preserve the scroll state and internal navigation state of each tab when switching between them.
