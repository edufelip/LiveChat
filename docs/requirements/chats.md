# Chats Requirements

## R-016: Real-time Messaging
The system shall synchronize messages in real-time between participants using Firebase Firestore.

## R-017: Swipe Actions
The Inbox list shall support swipe actions for Pinning, Muting, and Archiving conversations.

## R-018: Read Receipts
The system shall display "Sent", "Delivered", and "Read" status indicators for all outgoing messages.

## R-019: Presence Update
The app shall update the user's online status in the database when the app enters the foreground.

## R-020: Privacy Constraints (Presence)
The system shall respect "Last Seen" privacy settings; if a user hides their status, they cannot see others' status.

## R-021: Message History
The system shall support paginated loading of conversation history to optimize performance.
