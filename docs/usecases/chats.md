# Chats Use Cases

## UC-011: Messaging
**As a** user,
**I want to** send and receive text messages in real-time,
**So that** I can communicate with my contacts.

## UC-012: Managing Conversations
**As a** user,
**I want to** pin, mute, or archive conversations by swiping,
**So that** I can keep my inbox organized.

## UC-013: Presence Tracking
**As a** user,
**I want to** see if my contact is "Online" or when they were last active,
**So that** I know if they are available to chat.

## UC-014: Typing Indicators
**As a** user,
**I want to** see when my contact is typing a message,
**So that** I know a response is coming.
