# Contacts Use Cases

## UC-018: Contact Syncing
**As a** user,
**I want to** sync my phone's contact list with LiveChat,
**So that** I can automatically find friends who are already using the app.

## UC-019: Inviting Friends
**As a** user,
**I want to** invite friends who are not yet on LiveChat,
**So that** they can join the platform and chat with me.

## UC-020: Starting Conversations
**As a** user,
**I want to** tap on a contact's name,
**So that** I can instantly start a new conversation with them.
