# Chat Features Use Cases

## UC-015: Sending Images
**As a** user,
**I want to** share photos from my gallery or camera,
**So that** I can share visual moments with my contacts.

## UC-016: Voice Messaging
**As a** user,
**I want to** record and send voice messages,
**So that** I can communicate faster or hands-free.

## UC-017: Viewing Media
**As a** user,
**I want to** view images in full-screen and listen to audio messages with a progress bar,
**So that** I can clearly understand the shared media.
