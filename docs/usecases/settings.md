# Settings Use Cases

## UC-021: Profile Personalization
**As a** user,
**I want to** update my display name and status message,
**So that** others can recognize me.

## UC-022: Theme Selection
**As a** user,
**I want to** switch between Light and Dark mode,
**So that** the app's appearance matches my preference.

## UC-023: Managing Privacy
**As a** user,
**I want to** block specific contacts or hide my "Last Seen" status,
**So that** I can control my privacy on the platform.

## UC-024: Account Deletion
**As a** user,
**I want to** permanently delete my account and all associated data,
**So that** I can leave the platform completely.
